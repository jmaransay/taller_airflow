# UR Airflow Workshop #

## Deploy Airflow ##

Go to docker folder `cd docker` and run `docker compose up [-d]`

### Access Airflow UI ###

Go to [localhost:8080](localhost:8080) in your browser when everything is ready

- Username: `airflow`
- Password: `airflow`

## Upload own DAGs ##

Copy your DAG files in `docker/airflow/dags`
